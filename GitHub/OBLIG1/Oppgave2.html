Hva er en webserver?


En webserver et program eller maskin som lagrer og utleverer data på nett.
Denne dataen kan hentes fra protocols som heter "HTTP" og "HTTPS". HTTP står for "Hyper Text Transfer Protocol".
HTML står for "Hyper Text Markup Language" og protocols henter dette innholdet fra en webserver via "HTTP" eller "HTTPS".
HTTPS står for "Hyper Text Transfer Protocol Secure" og dette er en tryggere utgave av "HTTP".
En HTTPS krypterer og beskytter brukeren fra tyvlytting eller at noen endrer på data.
